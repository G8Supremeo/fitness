Suprimify Performance Lab

Real-life Health & Fitness Tracker for end users with smart-device sync, quick analytics, and behavior-focused progress intelligence.

Problem We Solve

  • Users have fragmented fitness data across wearables and apps.
  • Most trackers show logs but fail to provide actionable performance trends.
  • Manual data entry creates friction and lowers retention.

Core Product Capabilities

  • Authentication: register/login accounts
  • Real backend API with SQLite persistence and JWT sessions
  • Exercise logs: type, duration, calories, notes, date
  • Weekly/monthly goals with progress tracking
  • 30-day analytics chart (workouts + calories)
  • Streak tracking and reminder notifications

Smart Device Data Transfer

Live Device Connect

Apple Health, Fitbit, and Garmin can be connected from dashboard.

No manual upload required for regular updates.

Auto Sync Pipeline

Server sync endpoint pulls activity for connected providers every few minutes.

Analytics refresh after each sync, with source-tagged log records.

Technical Requirements Coverage

  • useState + useReducer for form/state control
  • useEffect for derived metrics and reminders
  • useMemo for date-range filtering and analytics
  • Recharts integrated for time-series visualizations
  • React Router for Dashboard, History, Goals, Auth
  • Custom useStreak hook for consecutive activity days

Differentiators

  • Real backend API with JWT auth and SQLite persistence
  • Remember-username login flow and branded Suprimify UX
  • Auto device sync for connected providers (no repeated uploads)
  • Cross-metric charting (activity and calories together)
  • Reminder system for behavior consistency

Product Walkthrough (3 mins)

  1. Create account and log first workout
  2. Set weekly goal and show progress
  3. Import Fitbit/Apple template file
  4. Show import report and updated 30-day chart
  5. Open history page to confirm source-tagged entries

Roadmap to Production

  • OAuth connectors for Fitbit/Garmin APIs
  • Move from SQLite to managed Postgres with encrypted backups
  • Coach dashboards and weekly AI insights
  • Mobile app companion
