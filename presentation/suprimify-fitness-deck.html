Technical Stack

  • Frontend: React 19, React Router, Vite, Recharts
  • UI System: CSS Custom Properties, glassmorphism, light/dark theming via useTheme
  • Custom Hooks: useStreak, useBluetooth, useNotificationReminder, useToast
  • API: Express + Node.js, REST JSON, JWT-protected routes
  • Data Store: better-sqlite3 with auto-migration of users, logs, goals, sync events, reset tokens
  • Hardware: Web Bluetooth (Heart Rate GATT 0x180D + battery service)

End-to-End Data Flow

  1. User signs up → backend hashes password & issues recovery phrase + JWT.
  2. Bluetooth device pairs in-browser; HR samples stream into a recording session.
  3. "Stop & save" posts to /api/logs/bluetooth → log + sync_event recorded.
  4. Dashboard recomputes streaks, weekly minutes, calorie totals, and 30-day chart.
  5. Sync Timeline page polls /api/sync-events for per-device status.

Theming Engine
